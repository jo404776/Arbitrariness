# Arbitrariness
## Project Description
A site that updates weekly with 8 random albums for people who are looking to listen to different things.

## Intended User or Player
Anyone who is into music and wants to listen to things they might have not heard before.

## User / Player Goal
The user should be able to understand that the albums listed are randomly chosen by me and they can choose if they want to listen to any of them.

## Inspiration Interfaces
Pitchfork- Grid layout
The Quietus- Also grid layout + header

## Design Choices
- Layout: A grid layout made in CSS. A grid is used inside the right column to format the albums.
- Color: Blues
- Typography: Web fonts
- Visual hierarchy: Header catches the eye first, followed by left to right. Headers are in a different color from other text to create better visual hierarchy. Albums section takes up the most space to also improve visual hierarchy.
- Feedback states: Hovering over the links in the menu section turns the links blue and increases their size to let users know they could click them.
- Responsive design: Layout changes from 2 columns into stacked 1 column on smaller screens.
